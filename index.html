Automator script.

Copia el codigo del automator script en egg.sh y despues ejecutado como script bash bash egg.sh

Una vez creado tu eggdrop modifica tu config.

Ej: Si quieres el modulo de twitter necesitaras cargar los siguientes scripts agregandolos al final de tu eggdrop.conf:

source scripts/twitter.tcl

source scripts/sha1.tcl

source scripts/oauth.tcl

source scripts/base64.tcl

Una vez terminado, para correr tu bot, dentro de tu carpeta eggdrop ejecuta: ./eggdrop eggdrop.conf
